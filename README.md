# Киномир - Одностраничный сайт

## Цель проекта

Разработать одностраничный сайт (лендинг) для демонстрации фильмов с использованием HTML, CSS и JavaScript. Сайт должен быть интерактивным, адаптивным и опубликованным в интернете, с возможностью переключения тем оформления.

## Технологии

* **HTML:** Семантическая разметка.
* **CSS:** Стилизация, адаптивность (Flexbox/Grid), переменные для тем.
* **JavaScript:** Интерактивные элементы, динамическая загрузка контента.
